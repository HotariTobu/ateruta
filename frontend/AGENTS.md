# frontend

SolidJS SPA. Built with Bun, output as static files.

## Technologies

- SolidJS — Signal-based reactivity (no Virtual DOM)
- Tailwind CSS v4 — Styling
- Fuse.js — Fuzzy search for song titles
- MusicKit JS v3 — Apple Music playback & playlist search (loaded from CDN)

## Commands

@Makefile

## Screens

- Home — Create/join room
- Lobby — Player management, playlist selection, game settings
- Game — Song playback, answer input, scoring
- Result — Final score display
